3. Fitur Utama

  • Notifikasi Real-time: Manager dan RMFT menerima notifikasi jika masih menggunakan password default
  • Badge Indicator: Icon bell menampilkan badge merah dengan jumlah notifikasi
  • Status Tracking: Admin dapat melihat siapa saja yang sudah/belum mengubah password
  • Timestamp: Sistem mencatat kapan password terakhir diubah
  • Auto Refresh: Notifikasi di-refresh otomatis setiap 5 menit
  • Security: Password selalu di-hash dan timestamp dicatat untuk audit

4. Cara Penggunaan

Untuk Manager & RMFT:

  1. Login ke sistem
  2. Perhatikan icon bell di navbar (akan ada badge merah jika ada notifikasi)
  3. Klik icon bell untuk melihat notifikasi
  4. Klik "Ubah Password" untuk ke halaman profil
  5. Ubah password Anda
  6. Notifikasi akan hilang setelah password berhasil diubah

Untuk Admin:

  1. Buka menu "Akun" di sidebar
  2. Lihat kolom "STATUS PASSWORD" di tabel Manager dan RMFT
  3. Badge kuning (âš  Password Default) = belum mengubah password
  4. Badge hijau (âœ“ Sudah Diubah) = sudah mengubah password dengan tanggal
