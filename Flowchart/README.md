## ðŸš€ Quick Start

### Cara Tercepat (5 menit)
1. Buka file **`INSTRUKSI-LENGKAP.md`**
2. Pilih salah satu dari 3 cara:
   - **Cara 1**: Manual di Diagrams.net (Paling Fleksibel)
   - **Cara 2**: Import Mermaid (Paling Cepat)
   - **Cara 3**: Python Script (Untuk Developer)

### Cara Manual (Recommended)
1. Buka **`PANDUAN-BUAT-FLOWCHART-ADMIN.md`**
2. Ikuti step-by-step
3. Buka https://app.diagrams.net/
4. Buat flowchart sesuai panduan
5. Export sebagai PNG/PDF

### Cara Otomatis (Mermaid)
1. Buka https://app.diagrams.net/
2. Menu: Arrange > Insert > Advanced > Mermaid
3. Copy isi file `admin-flowchart.mmd`
4. Paste dan klik Insert
5. Edit dan export
